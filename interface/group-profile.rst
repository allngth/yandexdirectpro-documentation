Профиль для групп продуктов
===========================

.. figure:: ../img/impexp.png
    :align: center

Для перехода на страницу редактирования профиля групп продуктов, перейдите в "Магазин" => "Импорт/экспорт" => "Яндекс.Директ".

В этом разделе можно создать новый профиль или посмотреть статистику по объявлениям, которые были сгенерированны для данного профиля во вкладке "Отчет по выгрузке".
